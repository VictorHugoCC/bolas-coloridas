Classificação por Bolas Coloridas

Movimentos: 0

Instruções

Organize as bolas coloridas de modo que cada tubo contenha apenas bolas de uma única cor.

  • Clique em um tubo para selecionar uma bola.
  • Clique em outro tubo para mover a bola selecionada.
  • Só é possível mover uma bola para um tubo vazio ou para um tubo cujo topo tenha a mesma cor.
